# Button

This atom will let the developer add easily a card. By itself it only represent the surface where the other atoms will be putted in.

## You can use a primary button

<Preview>
  <Story name='withPrimaryVariant' height='100px'>
    <Button>Hello Button</Button>
  </Story>
</Preview>

## You can use a secondary button

<Preview>
  <Story name='withSecondaryVariant' height='100px'>

  </Story>
</Preview>
